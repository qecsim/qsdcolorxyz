qsdcolorxyz - qecsim color code variant extension

qsdcolorxyz is a Python 3 package that extends qecsim with additional components for the XYZ variant of the color code.

The Overview section describes the key differences between the XYZ and standard color codes. The API section details the code and decoder implementations. See the README at qsdcolorxyz for Installation, Usage and Developer Notes. See also the Citing section for details on citing qecsim and its dependencies.
