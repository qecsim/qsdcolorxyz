qsdcolorxyz - qecsim color code variant extension
=================================================

`qsdcolorxyz`_ is a Python 3 package that extends `qecsim`_ with additional
components for the XYZ variant of the color code.

.. _qsdcolorxyz: https://github.com/qecsim/qsdcolorxyz
.. _qecsim: https://github.com/qecsim/qecsim

The *Overview* section describes the key differences between the XYZ and
standard color codes. The *API* section details the code and decoder
implementations. See the README at `qsdcolorxyz`_ for *Installation*, *Usage*
and *Developer Notes*. See also the *Citing* section for details on citing
qecsim and its dependencies.
